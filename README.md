# ecommerce
Implementação de um sistema de e-commerce com Laravel

## Requisitos para trabalhar com Laravel
- PHP >= 5.5.9
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
